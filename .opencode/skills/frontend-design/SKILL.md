此技能指导创建独特且可用于生产环境的前端界面，避免出现通用的“AI 拼凑感”（AI slop）美学。实施真正可运行的代码，并特别关注美学细节和创意选择。

用户提供前端需求：要构建的组件、页面、应用程序或界面。他们可能会提供关于目的、受众或技术限制的上下文。

## 设计思维

在编码之前，理解上下文并致力于一个\*\*大胆（BOLD）\*\*的美学方向：

* **目的 (Purpose)**：该界面解决什么问题？谁在使用它？
* **基调 (Tone)**：选择一个极端的风格：极致极简、极繁主义的混乱、复古未来主义、有机/自然、奢华/精致、俏皮/玩具感、社论/杂志风、野兽派/原始、装饰艺术/几何、柔和/粉彩、工业/实用主义等。有很多风格可供选择。以此为灵感，但设计出一个符合美学方向的作品。
* **限制 (Constraints)**：技术要求（框架、性能、可访问性）。
* **差异化 (Differentiation)**：是什么让它**令人难忘**？人们会记住的一件事是什么？

**关键 (CRITICAL)**：选择一个清晰的概念方向并精确执行。大胆的极繁主义和精致的极简主义都有效——关键在于**意图（intentionality）**，而不是强度。

然后实现可运行的代码（HTML/CSS/JS, React, Vue 等），要求：

* 生产级且功能完备
* 视觉冲击力强且令人难忘
* 具有清晰美学观点的凝聚力
* 每个细节都经过精心打磨

## 前端美学指南

重点关注：

* **排版 (Typography)**：选择美观、独特且有趣的字体。避免使用通用字体（如 Arial 和 Inter）；相反，选择能提升前端美学的独特字体；意想不到的、有个性的字体选择。将独特的标题字体与精致的正文字体配对。
* **颜色与主题 (Color & Theme)**：致力于连贯的美学。使用 CSS 变量保持一致性。具有鲜明强调色的主色调胜过胆怯、均匀分布的调色板。
* **动效 (Motion)**：使用动画实现效果和微交互。优先考虑 HTML 的纯 CSS 解决方案。在可用时使用 Motion 库。专注于高影响力的时刻：一个精心编排的页面加载（带有交错显示 animation-delay）比分散的微交互更能带来愉悦感。使用滚动触发和令人惊喜的悬停状态。
* **空间构图 (Spatial Composition)**：意想不到的布局。不对称。重叠。对角线流。打破网格的元素。充足的负空间或受控的密度。
* **背景与视觉细节 (Backgrounds & Visual Details)**：创造氛围和深度，而不是默认为纯色。添加符合整体美学的上下文效果和纹理。应用创意形式，如渐变网格、噪点纹理、几何图案、分层透明度、戏剧性阴影、装饰性边框、自定义光标和颗粒叠加层。

**永远不要**使用通用的 AI 生成美学，如过度使用的字体系列（Inter, Roboto, Arial, 系统字体）、陈词滥调的配色方案（特别是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏上下文特定特征的千篇一律的设计。

创造性地解读并做出意想不到的选择，这些选择感觉是真正为上下文设计的。没有两个设计应该是相同的。在明暗主题、不同字体、不同美学之间变化。**永远不要**在不同代际之间收敛于常见的选择（例如 Space Grotesk）。

**重要 (IMPORTANT)**：使实现复杂度与美学愿景相匹配。极繁主义设计需要带有大量动画和效果的复杂代码。极简主义或精致设计需要克制、精确，并仔细关注间距、排版和微妙的细节。优雅来自于对愿景的良好执行。

记住：你 有能力完成非凡的创意工作。不要退缩，展示当你跳出框框思考并完全致力于独特愿景时，真正能创造出什么。
